=============
Quick Recipes
=============

Here are some ready-made recipes for you to get familiar with |HorizonGRound|.

The data used in these tutorials are packaged with the released distribution
and can be accessed with the help of the :func:`~.utils.get_test_data_loc`
function.  Alternatively, test data sets can be downloaded from
`here <https://github.com/MikeSWang/HorizonGRound/tree/master/horizonground/tests/test_data/>`_.
Note that these data sets are mostly taken from Palanque-Delabrouille
et al. (2016) [1]_.

.. [1]_ Palanque-Delabrouille N. et al., 2016. **A&A** 587, A41.
   [arXiv: `1509.05607 <https://arxiv.org/abs/1509.05607>`_]
